üöÄ Getting Started

üê≥ Docker Compose Deployment

# Clone the repository
git clone https://github.com/your-repo/mongodb-crud-api.git
cd mongodb-crud-api

# Start complete multi-instance deployment
docker-compose up -d

# Services started automatically:
# - 3 API instances (ports 3001, 3002, 3003) with coordination
# - Nginx load balancer (port 8080) with health checks
# - React frontend (port 3004) with management interface
# - MongoDB database (port 27017) with connection pooling
# - Redis coordination (port 6379) for distributed operations

# Access points:
Load Balancer: http://localhost:8080
Frontend:      http://localhost:3004
API Direct:    http://localhost:3001, 3002, 3003
